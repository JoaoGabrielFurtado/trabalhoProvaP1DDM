Como usar o programa:
Dar play no Android Studio > Se logar usando "unisanta/unisanta" > Cadastrar um local (caso algum campo esteja vazio, o programa não salva) > Para verificar a lista de locais cadastrados basta clicar em VER LISTA > É possível ver a lista de locais cadastrados / excluir um local / acessar a página web salva no cadastro
